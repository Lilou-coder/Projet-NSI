# Projet-NSI
Projet NSI avec Juliette CHEM-LENHOF, Alexandre EGUERRE, Lilou COLLE

Notre idée : un site éducatif pour les sixièmes, contenant l'entierté du programme mathématiques.
Il sera sous la forme d'un jeu, avec differents niveaux pour les differentes parties du programme. 

Vision du jeu : 
- un jeu de tire à la première personne dans un  vaisseau
- pour pouvoir tirer il faut
